# Example PDFs

This folder contains sample PDF files for testing the PDF Combiner application.

## Files

- `jabberwocky.pdf` - Lewis Carroll's "Jabberwocky" poem (public domain)
- `shakespeare_sonnets.pdf` - Shakespeare's Sonnets 18 and 116 (public domain)

## How to Test

1. Run the application with `npm run tauri dev`
2. Click "Add PDFs" and select both files from this folder
3. Arrange them in your preferred order using the drag handles
4. Click "Combine PDFs"
5. Choose a location to save the combined file
6. Open the output to verify both documents merged correctly
